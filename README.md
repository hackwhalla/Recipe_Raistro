## ✨ Overview

**Recipe Raistro** is an intelligent Agentic AI-powered nutrition platform that transforms dietary goals into personalized meal plans and recipes. Using **Google Gemini 2.5 Flash** as the core reasoning engine, it autonomously analyzes requirements, accesses nutritional databases, and generates customized recipes with precise macronutrient optimization.
